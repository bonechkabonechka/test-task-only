## Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/bonechkabonechka/test-task-only.git
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. Запустите проект в режиме разработки:
   ```bash
   npm start
   ```
